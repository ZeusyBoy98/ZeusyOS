# ZeusyOS
## About
ZeusyOS is a fake web-based OS made entirely for the enjoyment of the user. Try it out at https://os.zeusyboy.com
## Features
- Interactive setup
- Beautiful home screen
- 5 custom apps
- Amazing consistent design
- Plenty of easter eggs!
## Apps
- Landslide web browser, browse the very limited landslide content transfer protocol web
- Notepad, create save and delete notes
- Weather, see weather in your area if you provide your own api key
- Clock, use stopwatches and timers
- Sketch, make cool drawings
## Why
I'm making it for flavortown by hackclub with the goal of learning more advanced html, css, and js. 

## Running Locally
Simply press the green code button, download the zip, extract it, and then open the index.html file in any modern browser.
